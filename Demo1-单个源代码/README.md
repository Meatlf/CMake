## Demo1-单个源代码

该例为 CMake 的 HelloWorld。

执行步骤如下：

```shell
mkdir build

cd build

cmake ..

make

./Demo
```

**Q**：为什么要学 CMake？

**A**：方便于构建大型项目。

**Q**：CMake 的特点有哪些？

**A**：跨平台、能够管理大型项目、简化编译构建过程和编译过程（只需 cmake+make）、高效、可扩展。

**Q**：如何学习 CMake？

**A**：多实践。

**Q**：如何安装 CMake？

**A**：很简单的。
